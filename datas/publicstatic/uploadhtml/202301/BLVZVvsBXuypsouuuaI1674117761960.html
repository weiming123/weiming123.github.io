什么是 URI ？

  URI 的全称是：Uniform Resource Identifier，即：统一资源标识符，是一个用于标识某一互联网资源名称的字符串。

  什么是 URL ？

  URL 的全称是：Uniform Resource Locators，即：统一资源定位系统，是用于指定信息位置的表示方法。

  什么是 URN ？

  URN 的全称是：Uniform Resource Name，即：统一资源名，是通过名称来指定信息位置的表示方法。

  URI、URL、URN之间的区别和联系

  URI可以分为URL、URN或同时具备locators 和names特性的一个东西。URN作用就好像一个人的身份证，URL就像一个人的地址。换句话说：URN确定了东西的身份，URL提供了找到它的方式。
